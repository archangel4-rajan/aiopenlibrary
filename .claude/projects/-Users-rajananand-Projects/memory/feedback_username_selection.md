---
name: username-selection-ux
description: User wants a dedicated username selection page where users confirm their username, not just a pre-filled field
type: feedback
---

When suggesting usernames or auto-filling profile fields, always present it as a suggestion that the user must explicitly confirm. Build dedicated selection/confirmation flows rather than silently setting values.

**Why:** The user wants users to feel ownership over their identity. A dedicated page makes it feel intentional, not automated.

**How to apply:** For any profile-setup or onboarding flow, create a dedicated step/page rather than embedding it in settings. Present suggestions but require explicit confirmation.
